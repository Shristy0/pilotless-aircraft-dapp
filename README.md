Pilotless Aircraft Monitoring DApp

 Overview
This project implements a blockchain-based decentralized application (DApp) for monitoring and logging pilotless aircraft operations.

Technologies Used
- Solidity
- Hardhat
- OpenZeppelin
- Ethereum Sepolia TestNet
- React (Frontend)
- Ethers.js

 Features
- Aircraft registration
- Flight logging
- Blockchain event emission
- TestNet deployment

 Installation

1. Clone repository:
   git clone <repo-url>

2. Install dependencies:
   npm install

3. Compile contracts:
   npx hardhat compile

4. Deploy to Sepolia:
   npx hardhat run scripts/deploy.js --network sepolia
